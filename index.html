Quick Entry
Paste JSON Import

Paste a peakos.health.v1 payload from your Shortcut or clipboard.

Set up the Shortcut once. Every morning it reads Apple Health, encodes your data, and opens Peak OS with it pre-loaded — no tapping required.

  1. 1
    Open Shortcuts → tap + → name it Peak OS Morning Sync.
  2. 2
    Add Get Health Sample actions for: Weight, Body Fat %, HRV, Resting Heart Rate, Sleep Analysis, Active Energy Burned, Step Count.
  3. 3
    Add a Text action and build the JSON: 
    {"schema":"peakos.health.v1","date":"[Date]","body":{"weightLbs":[Weight],"bodyFatPct":[Body Fat %]},"health":{"hrvMs":[HRV],"restingHr":[Resting HR],"sleepHours":[Sleep Hours],"activeCalories":[Active Energy],"steps":[Steps]}}
  4. 4
    Add Encode Base64 on the Text (URL-safe mode if available).
  5. 5
    Add Open URL: 
    [your-app-url]?import=[Base64 Encoded Text]
  6. 6
    In Automation, schedule at your morning time with Run Immediately enabled.
The body.hume block and every health field are optional — missing fields are silently skipped.
